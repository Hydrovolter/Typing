Typing Test User Guide - 06/09/22

Redo

Hold "shift" while pressing the Redo button to restart the typing test using the same word list

Press "esc" to restart a typing test

Language

Type the language code in the text box then hit:

Windows: alt + l
Mac: cmd + ctrl + l
Linux: super + ctrl + l or alt + l

Available Languages:

english
english1000
italian
german
spanish
chinese
korean
polish
swedish
dots
symbols
numbers
punjabi
french
portuguese

Typing Mode

Type the mode code in the text box then hit:

Windows: alt + m
Mac: cmd + ctrl + m
Linux: super + ctrl + m or alt + m

Available Modes:

wordcount
time

Punctuation

Type "true" or "false" then hit:

Windows: alt + p
Mac: cmd + ctrl + p
Linux: super + ctrl + p or alt + p

To activate/deactivate punctuation

Theme Menu

In this menu you can see all the available themes to choose from, along with a small preview.

Hit "escape" or click "back" to exit

Calculations

WPM (Words Per Minute): total number of characters (including spaces) of words you correct divided by five then divided by the time starting from first character typed

ACC (Accuracy): total number of characters (including spaces) of words you correct divided by all character in the list of words
